Welcome to PPL, the FI-WARE Data Handling GE reference implementation.

This project hosts the source code of PPL. This code has been developed thanks to the support of the FI-WARE project, an initiative funded by the European Commission. Please refer to the FI-WARE website for more information.

The Data Handling GE mainly focuses on revealing certain attributes according to specific privacy and security conditions. It supports integrated data handling, in particular through two-sided detailed data handling, that takes into account specific preferences/policies expressed using the PPL language, based on XACML. Data usage purpose must always be declared, as it is a relevant part of the policy that must be expressed, as well as downstream usage, i.e., whether one can disclose collected data with third parties. The PPL language supports the enforcement of a number of obligations, that are bound tightly to data. For instance, one can impose a specific retention period, as well as the production of user's notifications and/or logging under certain conditions.

PPL is an implementation of the Data Handling GE specification.

Why do I need PPL?

When private and sensitive data is sent and stored into external web or cloud servers, there is no real control by the data owner. This lack of control is due to the absence of mechanisms and methods that provides access and usage control to the stored data during its life time. PPL provides the framework and the necessary tool to give the control to the data owner by imposing obligations on the data and restrictions.

Quick Reference Guide

PPL can be used with a standard web browser, through a simple web application interface:

  • http://datahdl.testbed.fi-ware.eu:8080/ppl-webapp/ (Secure File Store Service use case)

PPL offers a REST API that is documented at the Data Handling REST API page.

The REST API of PPL is directly accessible at the following REST endpoint: http://datahdl.testbed.fi-ware.eu:8080/ppl-rest/

It is possible to begin using PPL from the Data Subject ReSTful API, using a simple method with no input parameters. The Data Handling GE API page contains more information about methods exposed by PPL service of Data Handling GE.
